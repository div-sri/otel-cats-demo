# otel-cats-demo

### Overview

This repository contains a Proof of Concept (PoC) focused on integrating OpenTelemetry tracing into a Scala application.
The application is built using cats-effect, http4s, and doobie, and demonstrates how to add tracing capabilities to an existing codebase.

The main goal of this PoC is to provide a simple, practical example of how to set up and use OpenTelemetry tracing in a Scala backend service.
The application logic consists of basic CRUD operations and is intentionally straightforward—the emphasis is on tracing integration rather than business logic complexity.

### Tech Stack
- Scala 3
- cats-effect
- http4s
- doobie
- opentelemetry
- otel4s
- mysql
- docker
